# Task 01: Data Cleaning and Visualization

## Overview
This project performs end-to-end data cleaning and exploratory data analysis (EDA) on an open-source dataset using Python. It focuses on transforming raw, messy data into a clean, analysis-ready format and uncovering patterns through visualization.

## Objectives
- Handle missing values, duplicates, and incorrect data types
- Detect and address outliers
- Perform exploratory data analysis to identify trends and correlations
- Build clear, insightful visualizations

## Tech Stack
- **Python**
- **Pandas** & **NumPy** – data manipulation and cleaning
- **Matplotlib** & **Seaborn** – static visualizations
- **Plotly** (optional) – interactive visualizations

## Workflow
1. **Data Loading** – Import the raw dataset
2. **Data Cleaning**
   - Handle missing/null values
   - Remove duplicate records
   - Fix inconsistent data types
   - Treat outliers
3. **Exploratory Data Analysis**
   - Histograms for distribution analysis
   - Bar charts for categorical comparisons
   - Box plots for outlier detection
   - Correlation heatmaps for feature relationships
4. **Insights** – Summarize key trends found in the data

## Key Learnings
Data preprocessing, EDA techniques, and visualization skills foundational to any data science workflow.
